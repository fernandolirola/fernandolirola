## 🚀 Estatístico e atualmente Odds Compiler @ BetMGM | Betting Tech & Data Modeling
Atuo no desenvolvimento de soluções de alta performance para o ecossistema de apostas esportivas, focando em cálculo, modelagem estatística e automação de processos

### 🛠️ No que me destaco:
- **Arquitetura de Dados Esportivos:** Construção de pipelines robustos para extração e processamento de dados em tempo real (Scraping & APIs).
- **Modelagem Quantitativa:** Implementação de motores de cálculo baseados em distribuições estatísticas (Poisson, Negative Binomial, Hurdle) para precificação de mercados.
- **Interfaces para Trading:** Desenvolvimento de dashboards otimizados (UI/UX) para tomada de decisão rápida e compilação de odds.
- **Automação de Workflow:** Criação de ferramentas que transformam dados brutos em arquivos prontos para integração em sistemas de automação de apostas.
